 Api Dekati Rumah Warga Cemoro Sewu, Penampakannya

<isi>
 Solopos.com, KARANGANYAR ? Kebakaran lereng <objek>Gunung Lawu</objek> kian meluas. sore ini, (25/10/2015), api mendekati permukiman penduduk Cemoro Sewu, Tawangmangu, jalan tembus Tawangmangu-Magetan. Dari foto diterima redaksi Solopos.com, api menjalar turun dari perbukitan deretan rumah warga. Api dari jalan raya bukit. itu, kebakaran hutan <objek>Gunung Lawu</objek> ribuan warga Desa Gondosuli Blumbang, Tawangmangu, <kabupaten>Karanganyar</kabupaten>, terancam gangguan pernafasan asap kebakaran. Pantauan Solopos.com Pos Cemoro Sewu, Plaosan, Magetan, (24/10/2015), terdapat enam titik api <objek>Gunung Lawu</objek>. Kepulan asap membumbung ke angkasa menandakan api melumat tanaman rumput hutan Lawu. Banyaknya titik api Gunung Lawu tertutup asap siang hari. Lawu terkesan misterius. Sesekali Lawu menampakkan wujud aslinya gagah. Angin asap kebakaran mengarah ke notabene wilayah Bumi Intanpari, ke selatan. api mengamuk Pos I Cemoro Sewu, (22/10/2015) lalu, padam (23/10/2015). Beruntung, api berhasil dikendalikan menjalar ke permukiman warga Pos Cemoro Sewu. <objek>Warga Cemoro Sewu</objek>, Plaosan, Sri Lestari, 31, ditemui Espos, khawatir munculnya api Pos I.
