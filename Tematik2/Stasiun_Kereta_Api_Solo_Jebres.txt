 Stasiun Kereta Api Solo Jebres

<isi>
 Stasiun Kereta Api Solo Jebres, selanjutnya disebut Stasiun Jebres, stasiun kereta api manajemen PT. Kereta Api Indonesia (Persero) Daerah Operasi (Daop) 6 Yogyakarta ketinggian +97 m permukaan laut. Stasiun Jebres terletak Jalan Ledoksari No. 1 Kelurahan Purwadiningratan, Kecamatan Jebres, Kota Surakarta, Provinsi Jawa Tengah. Lokasi stasiun sebelah timur Pasar Ledoksari. Hadirnya jalur kereta api melintas Kota Solo terlepas dari eksistensinya itu. kota kerajaan (Kraton Kasunanan Pura Mangkunegaran), pemerintah Hindia Belanda melancarkan diplomasi kerajaan Kota Solo tersebut. kemudian, kepentingan diplomasi berubah kepentingan angkutan manusia dimanfaatkan mengangkut komoditas perkebunan tembakau, gula, kopi lain-lain. 1847, Vorstenlanden (Surakarta Yogyakarta) terdapat 47 perkebunan. antaranya pabrik gula Gubernemen (pemerintah), perkebunan tembakau, perkebunan kopi. Praja Mangkunegaran industri gula, Pabrik Gula Colomadu Tasikmadu, Kasunanan Surakarta industri gula Pabrik Gula Gondang Winangoen komoditas tembakau vorstenlanden bahan baku cerutu dari Klaten. Komoditas-komoditas komoditas laku dibutuhkan dunia masanya pasar Eropa dibutuhkan alat transportasi mengangkut jumlah, kereta api. catatan sejarah, Stasiun Jebres dibangun 1884 lahan Kraton Kasunanan Surakarta, pengerjaannya perusahaan kereta api Pemerintah Hindia Belanda, Staats Spoorwegen (SS). Sehingga, stasiun dikenal Stasiun Staats Spoorwegen. transportasi penumpang mengangkut komoditas perkebunan, tercatat Stasiun Jebres dahulu Gubernur Jenderal Hindia Belanda hendak Sri Susuhunan Paku Buwono X (PB X). stasiun terdapat Kota Solo, Stasiun Purwosari, Stasiun Solo Balapan, Stasiun Solo Kota, Stasiun Jebres terbilang megah gaya arsitektur Indische Empire. Bangunannya berbentuk persegi simetris jendela melengkung pintu menuju ke hall stasiun fasad detail dipengaruhi aliran Neo-Klasik. Interiornya indah hadirnya pilar-pilar bergaya Corynthian Yunani jeruji besi jendelanya bergaya Art Nouveau. Kini, stasiun luas 1.631 m² bernomor register 164/06/SK/BD berfungsi stasiun menaikkan menurunkan penumpang kereta api kelas ekonomi dari Solo menuju Jakarta Bandung dari Solo menuju ke Surabaya sebaliknya. bangunan tua penuh sejarah ini, Stasiun Jebres ditetapkan bangunan cagar budaya (BCB) Walikotamadya Kepala Daerah II Surakarta Nomor: 646/116/I/1997 cagar budaya No. 13-25/C/Jb/2012.
