 Sejarah Kereta Api Indonesia

<isi>
 Dunia perkeretaapian Indonesia berjaya semenjak jaman penjajahan Belanda dahulu. sekarang, beroperasi jalur transportasi publik, angkutan barang, kereta wisata, dijadikan museum kereta api. menyedihkan, malah terawat, dilestarikan punah. Berikut Sharing Sini Stasiun Kereta Api Tertua Indonesia. 1. Stasiun Semarang Gudang / Tambaksari (1864). Stasiun dibangun 16 1864 diresmikan Gubernur Jenderal Baron Sloet van de Beele. pengoperasian rute ini, pemerintah Belanda menunjuk Nederlandsch Indische Spoorweg Maatschappij (NIS), markas NIS dikenal Lawang Sewu. tepatnya 10 Agustus 1867 kereta meluncur kalinya stasiun ini. 2. Stasiun Semarang Tawang (1868). Stasiun Semarang Tawang (kode SMT) stasiun induk Tanjung Mas, Semarang Utara, Semarang melayani kereta api eksekutif bisnis. Kereta api ekonomi singgah stasiun ini. Stasiun stasiun kereta api tertua Indonesia Semarang Gudang diresmikan 19 Juli 1868 jalur Semarang Tawang ke Tanggung. Jalur lebar 1435 mm. 1873 jalur diperpanjang Stasiun Solo Balapan melanjut Stasiun Lempuyangan Yogyakarta. 3. Stasiun Lempuyangan Yogyakarta (1872). Stasiun Lempuyangan (kode LPN, +114 m) stasiun kereta api terletak Kota Yogyakarta, berjarak 1 km sebelah timur dari stasiun kota ini, Stasiun Yogyakarta. Stasiun didirikan 2 Maret 1872 melayani pemberhentian KA ekonomi melintasi Yogyakarta. Stasiun Lempuyangan beserta rel membujur dari ke timur perbatasan Kecamatan Gondokusuman Danurejan selatan. 4. Stasiun Buitenzorg /Bogor (1872). Stasiun Bogor dibangun perusahaan kereta api Pemerintah Belanda 1872 stasiun jalur Batavia-Buitenzorg (Jakarta-Bogor) dibuka 1873. Pembukaan jalur mempersingkat perjalanan Batavia-Buitenzorg (Jakarta-Bogor) kereta kuda melayani penumpang. meningkatnya penumpang hilir-mudik Bogor-Jakarta (Jakarta-Bogor) sebaliknya diperlukan ruang-ruang menampung orang. 1881 dibangun stasiun baru. ruangan stasiun terdapat prasasti didirikan 1881. Prasasti terbuat dari marmer persembahan karyawan ucapan selamat D Marschalk memasuki pensiun jasanya mengembangkan perkeretaapian Pulau Jawa. Arsitekturnya bergaya Eropa berlantai hiasan motif. geometrik awan, kaki-kaki singa relung-relung lantai. ini, Kusen pintu jendelanya utuh gaya khas. Lapangan didepannya konon bernama Wilhemina Park. Bangunan luasnya 5.955 m2 dibangun areal lahan seluas 43.267 m2 lokasi tepatnya Jalan Nyi Raja Permas Kelurahan Cibogor Kecamatan Bogor Tengah. 5. Stasiun Ambarawa (1873). Museum Kereta Api Ambarawa stasiun kereta api dialihfungsikan museum Ambarawa, Jawa Tengah, kelengkapan kereta api berjaya zamannya. kereta api uap lokomotif nomor 2502 2503 buatan Maschinenfabriek Esslingen menjalankan aktivitas kereta api wisata. Kereta api uap bergerigi unik dari tersisa dunia. antaranya Swiss India. koleksi-koleksi unik tadi, disaksikan macam lokomotif uap dari seri B, C, D CC (CC 5029, Schweizerische Lokomotiv und Maschinenfabrik) halaman museum. 6. Stasiun Kedungjati (1873). Stasiun Kedungjati (KEJ) stasiun kereta api terletak Kedungjati, Kedungjati, Grobogan. Stasiun terletak ketinggian +36 m dpl Daerah Operasi 4 Semarang. Stasiun Kedungjati diresmikan 21 Mei 1873. Arsitektur stasiun serupa Stasiun Willem I Ambarawa, beroperasi jalur KA dari Kedungjati ke Ambarawa, beroperasi 1976. 1907, Stasiun Kedungjati tadinya dibangun dari kayu diubah ke bata berplester peron berkonstruksi baja atap dari seng setinggi 14,65 cm. 7. Stasiun Solo Balapan (1873). Stasiun Solo Balapan (kode: SLO, +93 m) stasiun induk Kestalan Gilingan, Banjarsari, Surakarta menghubungkan Kota Bandung, Jakarta, Surabaya, Semarang. Stasiun didirikan jaringan kereta api kolonial NIS abad ke-19 (tepatnya 1873). 8. Stasiun Tuntang (1873). Stasiun Tuntang beropersi 21 Mei 1873. Stasiun Tuntang stasiun kereta api terletak Tuntang, Semarang. Stasiun Tuntang dari jaringan Ambarawa-Tuntang-Bringin-Kedungjati. pengumpan rute Semarang-Solo melewati Kedungjati. Jalur rel Semarang-Solo dibangun kepentingan ekomomis (perkebunan kehutanan) militer. sebabnya Kedungjati, hutan, ketimbang Tuntang tepi jalan raya Semarang-Salatiga-Solo. Stasiun Tuntang stasiun kelas III jalur ini. jalur menghubungkan Yogyakararta Kedungjati ini, stasiun dijadikan museum. Stasiun ditutup melayani kereta wisata Ambarawa-Tuntang rel rusak. stasiun melayani lori Ambarawa-Tuntang. 2009 dimulailah renovasi stasiun melayani kereta uap wisata lagi. Direncanakan jalur menuju Stasiun Kedungjati dihidupkan lagi. 9. Stasiun Aceh Tramp (1874). Station van de Atjehtram te Oeleëlheuë bij Koetaradja. (Stasiun tram dari Aceh ke Oeleëlheuë Koetaradja). Kereta api Aceh, zaman disebut Atjeh Tramp, dimaksud sarana komuter (transportasi) masyarakat kota mobilitas berpindah antar kota, jarak-dekat. Sejarah Atjeh Tram apda 26 1874 Gubernur Aceh daerah taklukannya memerintahkan menghubungkan demarkasi pelabuhan Ulee Lheue Kutaraja rel kereta api sepanjang 5 km lebar spoor (rel) 1,067 m. Namun, sayanganya 1982 Banda Aceh resmi memilik jalur perhubungan kereta api lagi. dikarenakan bersaing sarana transportasi jalan raya onderdil sulit dicari. 10. Stasion Purwosari (1875). Stasiun Purwosari (PWS) stasiun kereta api terletak Jl. Slamet Riyadi No. 502, Purwosari, Lawiyan, Surakarta. Stasiun terletak ketinggian +98 m dpl Daerah Operasi 6 Yogyakarta. Stasiun Purwosari dibangun 1875, stasiun tertua Surakarta. Pembangunannya ditangani NIS. Stasiun Purwosari wilayah Mangkunegara. 11. Stasiun Surabaya Kota / Semut (1878). Stasiun Surabaya Kota (SB) populer Stasiun Semut terletak Bongkaran, Pabean Cantikan, Surabaya. Letaknya sebelah Stasiun Surabaya Gubeng stasiun tujuan kota Surabaya dari jalur kereta api selatan pulau Jawa menghubungkan Surabaya Yogyakarta Bandung Jakarta. sejarahnya, Stasiun Surabaya Kota dibangun jalur kereta api Surabaya-Malang Pasuruan dirintis 1870. Tujuannya mengangkut bumi perkebunan dari daerah pedalaman Jawa Timur, khususnya dari Malang, ke Pelabuhan Tanjung Perak dibangun itu. diresmikan 16 Mei 1878. meningkatnya kereta api, 11 Nopember 1911, bangunan stasiun perluasan ke bentuknya ini. 12. Stasiun Malang Kotalama (1879). Stasiun Malang Kotalama (MLK) stasiun kereta api terletak Kecamatan Sukun, Malang. Stasiun ketinggian +429 m dpl Daerah Operasi 8 Surabaya. Stasiun stasiun KA selatan Kota Malang, tertua, dibangun 1879. Penambahan “Kotalama” dimaksudkan membedakan Stasiun Malang Kotabaru dibangun belakangan. Dari Stasiun Malang Kotalama terdapat percabangan rel menuju ke Dipo Pertamina. 13. Stasiun Ijo (1880). Stasiun Ijo (IJ) stasiun kereta api terletak sebelah Stasiun Gombong. administratif, stasiun Desa Bumiagung, Kecamatan Rowokele, Kabupaten Kebumen. stasiun persilangan, fungsi pengontrol terowongan jalur rel (disebut Terowongan Ijo) sisi timur stasiun ini. Pengelolaan stasiun terletak ketinggian +25 m dpl Daerah Operasi 5 Purwokerto. Stasiun dibangun pertengahan 1880-an jarang disinggahi kereta api. Stasiun berperon sisi jalur rel. 14. Stasiun Jember (1897). Stasiun Jember (+89m dpl) stasiun kereta api lintasan sejarah perkerataapian Indonesia. Stasiun dibangun 1897 Staats Spoorwegen (SS), perusahaan kereta api zaman pemerintahan Hindia Belanda, melayani transportasi bumi wilayah Jember sekitarnya. itu, Komoditas perkebunan gula, karet tembakau diangkut dari Stasiun Jember menuju Pelabuhan Panarukan wilayah Situbondo selanjutnya diangkut kapal api menuju Kota Rotterdam Belanda. ini, Stasiun Jember terletak Jl. Dahlia No. 2, Jemberlor, Patrang, Jember, stasiun wilayah Jawa Timur pusat PT. Stasiun Klaten (1903). 1903 stasiun Klaten dioperasikan NIS (Perusahaan Kereta Api Hindia Belanda), Jawa Tengah. Stasiun Klaten (KT) stasiun kereta api terletak Tonggalan, Klaten Tengah, Klaten. Stasiun terletak ketinggian +151 m dpl terletak Daerah Operasi 6 Yogyakarta. Stasiun terletak jalan by-pass melingkari Klaten 6 jalur KA.
