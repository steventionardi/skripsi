 Stasiun Solo Balapan

<isi>
 Meskipun tampak menarik, bangunan Stasiun Balapan Solo keliahatan megah masanya. halaman luas mobil motor parkir, becak dokar lalang. ini, bangunan megah tertutup bangunan baru, halamannya sesak kendaraan roda becak mangkal. Kenyamanan keasrian disekitar stasiunpun berganti padatnya arus lintas kereta banyaknya penumpang pergi. Banyaknya calo penumpang menambah kesemrawutan stasiun. Stasiun Solo Balapan stasiun induk kota Surakarta menghubungkan Kota Bandung, Jakarta, Surabaya, Semarang. Stasiun didirikan jaringan kereta api kolonial NIS abad ke-19 (tepatnya 1873) stasiun tertua Indonesia (setelah Stasiun Semarang Tawang). Bangunan stasiun dirancang Herman Thomas Karsten, arsitek kenamaan beraliran Indisch. Setasiun Solo Balapan emplasemen, Selatan. Emplasemen Selatan 5 sepur (track), emplasemen 7 sepur. Emplasemen Selatan umumnya dipakai pelayanan KA Penumpang, Emplasemen diperuntukkan pelayanan KA barang. Ke arah Timur, terdapat jurusan, rel arah ke menuju ke Semarang, rel ke timur menuju Surabaya. sisi timur setasiun terdapat Segitiga Pembalik (Wye) memungkinkan rangkaian kereta api berbalik arah seluruhnya prinsip langsir. Sisi-sisi Segitiga Pembalik memungkinkan kereta api dari timur (dari setasiun Solo Jebres) ke / ke Semarang setasiun Solo Balapan sebaliknya. Segitiga Pembalik terdapat Depo BBM Pertamina, rel masuknya dari sisi Segitiga Pembalik ini. kereta api stasiun ini: -Eksekutif: Argo Lawu ke Jakarta, Argo Dwipangga ke Jakarta, Argo Wilis ke Gubeng, Surabaya/Bandung, Turangga ke Bandung/Gubeng, Surabaya. -Eksekutif bisnis: Sancaka ke Yogyakarta/Gubeng, Surabaya, Lodaya ke Bandung. -Bisnis: Jayabaya Selatan, Senja Solo, Mutiara Selatan, Prambanan Ekspres, Pandanwangi. -Ekonomi: Gaya malam Selatan, Sri Tanjung, Matamarja. Stasiun Solo balapan inspirasi lagu congdut populer 1990-an dari Didi Kempot: Setasiun Balapan.
