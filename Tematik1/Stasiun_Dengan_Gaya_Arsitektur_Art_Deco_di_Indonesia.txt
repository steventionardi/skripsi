 Stasiun Gaya Arsitektur Art Deco Indonesia

<isi>
 Kereta api alat transportasi Indonesia pulau Jawa Sumatra. Kehadiran kereta api semenjak 1864 kala Gubernur Jenderal Hindia Belanda, L.A.J Baron Sloet van den Beele meresmikan rel kereta pertama. kereta api sarana transportasi difavoritkan. stasiun Pulau Jawa menyajikan bangunan heritage sentuhan gaya arsitektural artdeco (perpaduan gaya arsitektural Perang Dunia I Perang Dunia II) khas. stasiun menyajikan gaya arsitektural artdeco kental terlihat. Berikut stasiun ! Stasiun Solo Jebres. Solo Jebres stasiun kereta api daerah Purwadiningratan, Jebres, Surakarta. Stasiun dibangun Staatsspoorwegen 1884. Stasiun Jebres bangunan heritage cagar budaya boleh dipugar pemerintah setempat. Stasiun Solo Jebres difungsikan pemberhentian kereta ekonomi. Stasiun Bogor. Stasiun Bogor sibuk melayani penumpang KRL dari menuju ke Jakarta. Stasiun Bogor kisah sejarah dunia kereta api. Stasiun berdiri 1881. Stasiun ketinggian 246 meter permukaan laut aktif ini. Tampilan Stasiun Bogor gaya arsitektural art deco klasik ke bangunan heritage. Stasiun Belawan. NV Deli Spoorweg Maatschappij disebut DSM pionir membangun stasiun Balawan. Kala DSM membujuk Hindia Belanda membangun jalur kereta api mengantarkan produksi bumi. Tampilan stasiun dari Belawan klasik gaya arsitektural art deco. Tampilan klasik dari abad ke 19 perpaduan modern abad ke 20 lihat. Stasiun Padalarang. Stasiun Padalarang dibangun pembangunan jalur kereta api dibangun dari kota Cianjur menuju Bandung 1884. Stasiun sentuhan art deco kental. atap tinggi. dindingnya tembok tebal. Keunikan dilihat jendela dari besi brankas. Suasana interior benar-benar klasik. Tampilan heritage menawan.
