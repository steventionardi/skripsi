 Mengenal Sejarah Stasiun Solo Jebres

<isi>
 Stasiun Solo Jebres stasiun terletak sebelah timur dari Jalan Urip Sumoharjo. Stasiun terletak ketinggian +97 m manajemen PT Kereta Api Indonesia Daerah Operasi VI Yogyakarta. Dari dibangun berfungsi ini, Stasiun Solo Jebres memliki sejarah panjang. Dimulai hadirnya jalur kereta api melintas Kota Solo terlepas dari eksistensi politik itu. kota kerajaan (Kraton Kasunanan Pura Mangkunegaran), pemerintah Hindia Belanda melancarkan diplomasi kerajaan Kota Solo. kemudian, kepentingan diplomasi berubah kepentingan angkutan manusia dimanfaatkan mengangkut komoditas perkebunan tembakau, gula, kopi lain-lain. 1847, Vorstenlanden (Surakarta Yogyakarta) terdapat 47 perkebunan. antaranya pabrik gula Gubernemen (pemerintah), perkebunan tembakau, perkebunan kopi. Praja Mangkunegaran industri gula, Pabrik Gula Colomadu Tasikmadu, Kasunanan Surakarta industri gula Pabrik Gula Gondang Winangoen komoditas tembakau vorstenlanden bahan baku cerutu dari Klaten. Komoditas-komoditas komoditas laku dibutuhkan dunia masanya pasar Eropa dibutuhkan alat transportasi mengangkut besar, kereta api. Stasiun Solo Jebres diresmikan 1884 tepatnya 24 Mei 1884 lahan Kraton Kasunanan Surakarta. Peresmian bersamaan pembukaan jalur kereta api Solo Jebres-Surabaya Kota. itu, pengerjaan perusahaan kereta api Pemerintah Hindia Belanda, Staats Spoorwegen (SS). stasiun dikenal Stasiun Soerakarta, stasiun dikenal 'Solo Kraton' kode stasiun SK. transportasi penumpang mengangkut komoditas perkebunan, Stasiun Solo Jebres dahulu Gubernur Jenderal Hindia Belanda hendak Sri Susuhunan PakoeBoewono X (PB X). itu, stasiun keluarga raja Keraton Surakarta bepergian ke Batavia Surabaya kereta api. lagi, Stasiun Solo Jebres kerap Raja Keraton Mangkunegara diresmikan. Stasiun Solo Jebres semacam 'stasiun resmi' Keraton Mangkunegara. terdapat kereta raja gerbong jenazah tersimpan rapi kompleks tunggu raja terdapat Stasiun Jebres. Tercatat, Stasiun Jebres pemugaran diperbesar 1906. ornamen bangunan tetap dipertahankan aslinya diresmikan 1884. perkembangannya, stasiun pemugaran besar-besaran. Stasiun Jebres terbilang megah gaya arsitektur Indische Empire. Bangunannya berbentuk persegi simetris jendela melengkung pintu menuju ke hall stasiun fasad detail dipengaruhi aliran Neo-Klasik (budaya Yunani Kuno abad pertengahan). Interiornya indah hadirnya pilar-pilar bergaya Corynthian Yunani jeruji besi jendelanya bergaya Art Nouveau ala Belgia. Tujuan pembangunan Stasiun Jebres pemerintah Hindia Belanda pengangkutan berupa komoditas tanaman industri gula tembakau andalan wilayah Karesidenan Surakarta. Komoditas perkebunan dikirim dari Jebres menuju Pelabuhan Cilacap dikirim ke Eropa. perkembangannya, Kroya-Cirebon tersambung 1917, pengiriman dialihkan menuju Pelabuhan Tanjung Priok Batavia. Stasiun Solo Jebres istimewa kolonial. Stasiun Solo Balapan stasiun pemberangkatan kereta api unggulan (eksekutif bisnis), Stasiun Solo Jebres 'turun kasta' melayani kereta api lokal kereta api ekonomi jarak jauh.
